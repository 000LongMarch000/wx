一、什么是微淘秀？
微淘绣是一款打通微信和手机淘宝的链接生成工具，任意手淘链接（聚划算，淘抢购，天猫店铺）都可以生成可以生成微信直通淘宝的短连接，是淘宝商家微信端营销的利器。
二、帮助目录
1. 苹果系统帮助
2. 安卓系统帮助
3. 会员等级与链接
1. 苹果系统
1.1 淘宝app里面访问商品的详情页面，点击分享链接
1.2 分享按钮弹出层里面，点击复制
1.3 点击“微淘秀”的公众号的键盘，在输入框里面长按粘贴，然后点击发送，生成链接. ps: 复制公众号生成后的链接，可转发朋友圈访问
2. 安卓系统
2.1 淘宝app里面访问商品的详情页面，点击分享链接
2.2 分享按钮弹出层里面，点击复制
2.3 点击“微淘秀”的公众号的键盘，在输入框里面长按粘贴，然后点击发送，生成链接. ps: 复制公众号生成后的链接，可转发朋友圈访问
3. 会员等级及链接
3.1 点击“会员”菜单，可以看到会员购买按钮和会员等级的介绍
3.2 点击“查看链接”菜单，可以看到已经创建好的链接，以及链接的过期时间等信息
